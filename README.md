# Inkwell

**Author:** Federico Ceballos Torres

**Description:**  
Repository for publishing novels, short stories, and political/philosophical essays.  
This repository stores chapters, versions, and working notes.  
Each content type is governed by its own licensing policy.

---

## Repository structure

- `/novels/es/novel001/Capítulo-01.md` — novel chapters in Markdown
- `/fiction/` — short stories
- `/philosophy/` — philosophical essays
- `/essays/` — political and analytical essays
